Compiling and installing from the source code

In this section wee see the installation method from source code, Download here


Dependencies of qualitytest program

  • Qt5 developer files.
  • Opencv2.4 or any in the family 2.X. (this code currently has some incompatibilities with 3.X family.)

Compiling and installing the project in Gnu-Linux

To install the qualitytest program in the path /usr/local/bin you need use (in the directory of this README file) the next commands:

    qmake PREFIX=/usr/local
    make
    sudo make install


Run the application

To execute the qualitytest program you need use the next command:

    qualitytest
