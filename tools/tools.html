Tools

You can get additional tools to analyze the biospeckle phenomenon.

SpeckleTool

The speckletool program is an image acquisition tool designed to work with the dynamic speckle phenomenon.

SpeckleOnline

The speckleonline program is an image acquisition tool, in online mode, designed to work with the dynamic speckle phenomenon.

Speckle Quality Test

The qualitytest program is an image processing tool designed to make quality test of images in a dynamic speckle test.

Portable Speckle

The portablespeckle applicative (APP) is an image acquisition and processing tool designed to work with the dynamic speckle phenomenon in Android systems.
