Installing from a binary file in Gnu-Linux

In this section we see two installation methods from a binary file.


Installing the speckleonline program from a snap

The snap files presented here can be installed in any gnu-linux distribution that support snaps, currently these are Ubuntu, ArchLinux, Debian, Gentoo, Fedora, OpenSuse, OpenEmbedded, Yocto and OpenWrt. The snaps are packages auto supported so that these have all the information necessary to works, consequently the snaps files have a very large size.

Please attention:

  • This snap program only can write data in the user HOME directory!
  • The user manual can't be launched from the speckleonline program but the console output of speckleonline program shows the location of manual.
    • .

To install the last version of the speckleonline program from a snap file you need download it from here . All snap files have a filename with the name structure speckleonline_X.Y.Z_ARCH.snap, where X.Y.Z is the version, and ARCH is the architecture, typically this value is i386 or amd64, to 32 or 64 bits architectures respectively.

Thus, the installation process of speckleonline_0.1.0_amd64.snap, downloaded in the directory path ~/Downloads, will be:

    sudo snap install --dangerous ~/Downloads/speckleonline_0.1.0_amd64.snap
    sudo snap connect speckleonline:raw-usb
    sudo snap connect speckleonline:camera

Uninstall the speckleonline program from snap

    sudo snap remove speckleonline


Run the application

To execute the speckleonline program you need use the next command:

    speckleonline
